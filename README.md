## 🎵 Project Name: Spotify Clone
Spotify Clone is a modern full-stack music streaming web application that replicates the core Spotify experience. Users can discover featured artists, browse albums and tracks, create and manage personal playlists, and enjoy seamless music playback with a clean, responsive Bootstrap interface.

## Main Functions

### 1. Artists:
   Browse artist cards with photos and imported Spotify data (name, genres, bio).
   Click through to detailed artist pages.

### 2. Albums:
   Browse album cards with cover images, titles and release dates.
   View detailed album pages with full track listings, duration and "Play" / "Add to Playlist" buttons.

### 3. Tracks & Music Player:
   Stream individual tracks from album pages.
   Global bottom player with play/pause, progress bar and queue support.

### 4. Playlists:
   Create, view, edit and delete personal playlists (My Playlists).
   Add/remove tracks directly from album pages (authenticated users only).

### 5. Authentication:
   User registration and login with JWT.
   Protected routes for playlists.

### 6. User Interface:
   Modern, responsive design with Bootstrap 5 cards, tables, dropdowns and smooth navigation.

## 🖥️ Technologies Used
### Frontend: Angular 19 (Standalone Components + SSR) + Bootstrap 5 + Bootstrap Icons  
### Backend: Django 5.2 + Django REST Framework  
### Authentication: SimpleJWT (JWT)  
### API Integration: Spotify Web API (spotipy Client Credentials)  
### Database: SQLite3  
### Other: Reactive Forms, HttpClient with JWT Interceptor, Auth Guard, Player Service
